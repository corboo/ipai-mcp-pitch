Connect Nigel to Claude

Add an AI gardening expert to your Claude conversations. Seasonal advice, plant identification, garden planning — all in Nigel's voice.

⚡ Takes under 60 seconds
1

Open Claude's Connector Settings

Click the button below to go directly to the "Add Custom Connector" dialog in Claude.ai. You'll need to be logged in.

Open Claude Settings →
2

Paste the MCP URL

In the URL field, paste Nigel's MCP endpoint:

https://mcp.nigel.inceptionpoint.ai/mcp

Give it a name like "Nigel Thistledown" and click Add.

3

Start Chatting!

That's it. Nigel is now available in your Claude conversations. Try asking:

💬 "Ask Nigel what I should plant this month in my garden"

Claude will automatically call the Nigel MCP tools when you ask gardening questions. You can also say "Use Nigel" to explicitly invoke him.

1

Open Your Config File

Claude Desktop stores MCP server configs in a JSON file. Open it at:

macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json

If the file doesn't exist yet, create it.

2

Add the Nigel MCP Server

Paste this JSON into your config file (or merge it if you already have other servers):

{
  "mcpServers": {
    "nigel-thistledown": {
      "url": "https://mcp.nigel.inceptionpoint.ai/mcp"
    }
  }
}
3

Restart Claude Desktop

Quit and reopen Claude Desktop. You should see Nigel's tools (🌱 seasonal advice, 🔍 plant info, etc.) available in your chat.

✅ That's it! Nigel is now a permanent part of your Claude Desktop. No need to reconnect each time.
